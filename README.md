# Hangman Game

## 📌 Project Description
This is a simple Hangman Game made using Python.

## 🚀 Features
- Random word selection
- Limited attempts (6 wrong guesses)
- Console-based game
- Input validation

## 🛠 Technologies Used
- Python
- Random Module
- While Loop
- If-Else
- Strings
- Lists

## ▶️ How to Run

```bash
python hangman.py
```
