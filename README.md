# Hangman Game

## Description
This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

## Features
- Random word selection
- 5 predefined words
- Maximum 6 incorrect guesses
- Console-based game
- Input validation

## Technologies Used
- Python
- Random Module
- While Loop
- If-Else
- Strings
- Lists
